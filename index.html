Two robots were built and programmed in C. The first one was a design similar to the year before where we used the iRobot roomba and built a claw on it out of legos and metal parts. Automated Chess Board Design

I designed an automated chess board from scratch using Solidworks. The XY table performs the displacements of the electromagnetic trolley. The trolley motorization is based on the Core XY concept. Two stepper motors with two belts are used to move the trolley following the X and Y axes. 20x20 aluminum rails were used and connected together by 3D printed parts specifically designed for the board.
If only one motor turns a diagonal displacement will be generated. Changing the motor rotation direction will give an inverted displacement. If two motors turn in the same direction a horizontal displacement will be generated. If two motors turn in opposite directions a vertical displacement will be generated.

Turtlebot3 Navigation Control

This above simulation in RViz shows the turtlebot3 robot navigating to two random positions.
I used a SLAM node (gmapping) to build a map of the robot's environment which is unknown to it while simultaneously estimating its position and orientation within that map.
The robot is equipped with sensors like 2D LiDAR scanners and a camera, which are essential create a map of a surrounding. Withing the package of this simulation, I created a launch file that contains two nodes: the first one intinalizes the position of the robot and the second one makes it move to its two different positions sequentially commanding the TurtleBot to navigate to specific coordinates (x, y, and w).
